Tipo de Input Ejemplo Breve descripción
Button Es un botón.
Checkbox Te deja elegir 0, 1 o varias de opciones limitadas.
Color Genera un selector de colores.
Date Permite elegir una fecha.
Datetime-local Permite elegir una fecha y hora.
Email Permite ingresar un e-mail.
File Permite ingresar un archivo de tu computadora.
Hidden Oculta el campo de input.
Image Permite elegir una imagen como botón de submit.
Month Permite elegir un mes.
Number Permite ingresar un número.
Password Permite ingresar una contraseña (y la oculta).
Radio

Permite elegir sólo una de varias opciones.
Range Crea un slider para elegir un valor.
Reset Permite deshacer todas las elecciones previas.
Search Genera un campo de búsqueda.
Submit Genera el botón para enviar el formulario.
Tel Permite ingresar un número de teléfono.
Text Permite ingresar un texto.
Time Permite ingresar una hora.
URL Permite ingresar una dirección web.
Week Permite seleccionar una semana.
